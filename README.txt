1.overflow:hidden对float的影响
通常：因为float元素脱离了文档流，所以包围它的div不占据空间（height和width都
为默认0的前提下），代码和显示效果如下：
@Code
.new { background-color : gray; border: solid 1px black; }
.news img { float : left; }
.news p { float : right; }
<div>
	<img src="" alt=""/>
	<p>Some text</p>
</div>	
            ===================================
            |``````||`````````````````````````|
            |______||_________________________|
               [最上面为div，下面是float元素]
如果想让float元素看起来包含在div中，此时窝们可以修改一下：
.clear { clear : both; }
在p元素下面添加：
<br class="clear" />
看上去img和p元素又在div中了：
